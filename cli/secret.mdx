Commands for managing Secret Manager vaults and secrets.

## secret

Manage Secret Manager vaults and secrets.

**Usage**

```bash
tailor-sdk secret [command]
```

**Commands**

| Command | Description |
| ------- | ----------- |
| `secret vault` | Manage Secret Manager vaults |
| `secret create` | Create a secret in a vault |
| `secret update` | Update a secret in a vault |
| `secret list` | List all secrets in a vault |
| `secret delete` | Delete a secret in a vault |

## secret vault

Manage Secret Manager vaults.

**Usage**

```bash
tailor-sdk secret vault [command]
```

**Commands**

| Command | Description |
| ------- | ----------- |
| `secret vault create` | Create a new Secret Manager vault |
| `secret vault delete` | Delete a Secret Manager vault |
| `secret vault list` | List all Secret Manager vaults in the workspace |

### secret vault create

Create a new Secret Manager vault.

**Usage**

```bash
tailor-sdk secret vault create [options] <name>
```

**Arguments**

| Argument | Description | Required |
| -------- | ----------- | -------- |
| `name` | Vault name | Yes |

**Options**

| Option | Alias | Description | Required | Default |
| ------ | ----- | ----------- | -------- | ------- |
| `--workspace-id <WORKSPACE_ID>` | `-w` | Workspace ID | No | - |
| `--profile <PROFILE>` | `-p` | Workspace profile | No | - |

**Example**

```bash
tailor-sdk secret vault create my-vault
```

### secret vault delete

Delete a Secret Manager vault.

**Usage**

```bash
tailor-sdk secret vault delete [options] <name>
```

**Arguments**

| Argument | Description | Required |
| -------- | ----------- | -------- |
| `name` | Vault name | Yes |

**Options**

| Option | Alias | Description | Required | Default |
| ------ | ----- | ----------- | -------- | ------- |
| `--workspace-id <WORKSPACE_ID>` | `-w` | Workspace ID | No | - |
| `--profile <PROFILE>` | `-p` | Workspace profile | No | - |
| `--yes` | `-y` | Skip confirmation prompts | No | `false` |

**Example**

```bash
tailor-sdk secret vault delete my-vault
```

### secret vault list

List all Secret Manager vaults in the workspace.

**Usage**

```bash
tailor-sdk secret vault list [options]
```

**Options**

| Option | Alias | Description | Required | Default |
| ------ | ----- | ----------- | -------- | ------- |
| `--json` | `-j` | Output as JSON | No | `false` |
| `--workspace-id <WORKSPACE_ID>` | `-w` | Workspace ID | No | - |
| `--profile <PROFILE>` | `-p` | Workspace profile | No | - |

**Example**

```bash
tailor-sdk secret vault list
```

## secret create

Create a secret in a vault.

**Usage**

```bash
tailor-sdk secret create [options]
```

**Options**

| Option | Alias | Description | Required | Default |
| ------ | ----- | ----------- | -------- | ------- |
| `--workspace-id <WORKSPACE_ID>` | `-w` | Workspace ID | No | - |
| `--profile <PROFILE>` | `-p` | Workspace profile | No | - |
| `--vault-name <VAULT_NAME>` | `-V` | Vault name | Yes | - |
| `--name <NAME>` | `-n` | Secret name | Yes | - |
| `--value <VALUE>` | `-v` | Secret value | Yes | - |

**Example**

```bash
tailor-sdk secret create --vault-name my-vault --name api-key --value "secret-value"
```
